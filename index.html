jquery.values-clarification.js

jQuery library to turn standard tables into an interactive decision-making tool.

Download this project as a .zip file Download this project as a tar.gz file

With just a little javascript and css, turn tables like this:

Name Description Label Value surgery1 surgery2
Avoiding a colostomy Avoiding a colostomy colostomy 40 1.0 -1.0
Avoiding a death Avoiding a death death 60 -0.5 2.0
Name Description Label Color
Surgery 1 Surgery 1 surgery1 #330066
Surgery 2 Surgery 2 surgery2 #999966

into an interactive decision-making tool like this:
